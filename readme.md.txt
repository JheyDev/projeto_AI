# Projeto Ia Generativa com Copilot e OpenAi

## Descrição
Este projeto consiste em reconhecer texto em imagens utilizando [Copilot e OpenAi].

## Estrutura do Repositório
- `inputs/`: Contém as imagens utilizadas.
- `output/`: Contém os resultados do reconhecimento de texto.

## Processo
1. Carreguei as imagens na pasta `inputs`.
2. Utilizei Copilot e OpenAi para realizar o reconhecimento de texto.
3. Salvei os resultados na pasta `output`.

## Insights
- Aprendi que a integração de IA generativa com ferramentas de reconhecimento de texto pode abrir novas possibilidades para automação e análise de dados.
- Descobri que quanto mais detalhado a pesquisa, mais eficiente o resultado.
